# NUC8i7BEH-macOS
My custom configurations for Hackintosh on NUC8i7BEH.

## Hardware

- CPU: Intel® Core™ i7-8559U Processor (8M Cache, up to 4.50 GHz)
- Memory: Curcial DDR4 2666 16GB
- M.2 SSD: WD Black SN750 500G NVMe
